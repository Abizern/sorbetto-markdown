# SorbettoMarkdown

SorbettoMarkdown is a Markdown-parsing plugin for [Sorbetto][sorbetto].

## Usage

```swift
import Sorbetto
import SorbettoMarkdown

try! Sorbetto()
    .using(Markdown())
    .build())
```
